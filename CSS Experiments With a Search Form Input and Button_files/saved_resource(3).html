                                        Thanks for a great tutorial! Very helpful

                                        However it works properly only in Crome. Here is a couple things I found:
                                        1. <input type="search"> doesn't apply styles in Safari, so I would go with type="text";
                                        2. FF requires "left: 0" for ".icon". Without it - icon just stays on the right;
                                        3. "input#search {box-sizing: border-box;}" needs to be added. Without it - form stretches, which makes 2nd example a wider rectangle;
                                        4. In second example placeholder is showing up a bit (everywhere except Chrome). Not sure how to deal with it without JS
